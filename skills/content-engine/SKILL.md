# Content Engine

## Purpose
Help the user create high-quality written content.

## Supported Formats
1. **Blog Post** — 800-2000 words, with headings and structure
2. **Social Media** — Platform-specific (Twitter, LinkedIn, Instagram)
3. **Newsletter** — Email-friendly with sections and CTAs

## Process
1. Ask for the topic or accept a brief
2. Search memory for the user's writing style preferences
3. Generate an outline first, then write the draft
4. Save to a markdown file

## Style Guide
- Match the user's voice (check USER.md)
- Use active voice, keep paragraphs short
- Include a compelling hook
- End with a clear call-to-action
